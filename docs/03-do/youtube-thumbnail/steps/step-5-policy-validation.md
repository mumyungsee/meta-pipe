---

## 2. 품질 검증 (`quality_checker.py`)

| 항목 | 기준 | 실패 시 |
| --- | --- | --- |
| 파일 크기 | <= 2MB | 차단 |
| 해상도 | 최소 640x360, 권장 1280x720 | 경고/차단 |
| 포맷 | JPEG 또는 PNG | 차단 |
| 비율 | 16:9 (1.7~1.85) | 차단 |

---

## 3. 정책 검증 (`policy_checker.py`)

### 3.1 금지 콘텐츠 → 차단

성적 암시, 폭력, 혐오 관련 키워드 감지 시 업로드 차단.

- 3 Strike 규칙: 90일 내 3회 위반 → 채널 종료
- 키워드 목록은 `BLOCKED_KEYWORDS`에 정의

### 3.2 클릭베이트 → 경고

과장된 표현 패턴 감지 시 경고 (업로드는 허용).

- "충격", "경악", "절대", "100%" 등
- 패턴 목록은 `CLICKBAIT_PATTERNS`에 정의

### 3.3 섬네일-제목 관련성 → 경고

섬네일 텍스트와 영상 제목 간 공통 문자가 없으면 클릭베이트 의심 경고.

### 3.4 이미지 안전 검증 (향후)

Google Vision SafeSearch API 연동 예정. 현재는 텍스트 기반 검증만 수행.

## 4. 사용법

```bash
python -m src.validator.cli \
  --image output/thumb.png \
  --title "AI로 10분만에 앱 만들기" \
  --text "AI 앱 제작"
```

출력 예시:
```
품질 검증: 통과 (5/5)
  [PASS] 파일 존재: output/thumb.png
  [PASS] 파일 크기: 0.85MB
  [PASS] 포맷: PNG
  [PASS] 해상도: 1280x720
  [PASS] 비율: 1.778 (16:9)

정책 검증: 통과
  문제 없음

종합: 업로드 가능
```

---

## 5. 검증 규칙 커스터마이즈

- `BLOCKED_KEYWORDS`: 금지 키워드 추가/삭제
- `CLICKBAIT_PATTERNS`: 클릭베이트 패턴 조정
- 채널 특성에 맞게 조정 필요 (예: AI 채널에서 "충격" 사용 허용 여부)

## 6. 완료 기준

- [ ] 정상 섬네일 → "업로드 가능" 출력
- [ ] 금지 키워드 포함 시 → "업로드 차단" 출력
- [ ] 클릭베이트 패턴 감지 시 → 경고 표시
- [ ] 규격 미달 이미지 → 해당 항목 FAIL 표시
