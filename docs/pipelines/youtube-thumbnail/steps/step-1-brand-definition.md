## 2. 브랜드 컬러

### 2.1 컬러 팔레트

| 역할 | 컬러 | HEX | 용도 |
| --- | --- | --- | --- |
| **Primary** | Electric Blue | `#2D7FF9` | 메인 강조색. AI/테크 느낌 + 밝고 신뢰감 |
| **Secondary** | Vivid Yellow | `#FFD600` | 텍스트 하이라이트, 주목 유도. 에너지/활기 |
| **Dark** | Deep Navy | `#0D1B2A` | 텍스트 배경, 오버레이. YouTube 다크 피드에서 대비 |

### 2.2 컬러 선정 근거

- **Electric Blue `#2D7FF9`**: AI/테크 채널의 업계 표준 톤. 밝고 활기차면서도 전문성 전달. YouTube 다크 피드(#0F0F0F)에서 강한 대비
- **Vivid Yellow `#FFD600`**: 섬네일에서 가장 눈에 띄는 색상 중 하나. Blue와 보색 관계로 텍스트 가독성 극대화. "주의! 이거 꼭 봐야 해" 느낌
- **Deep Navy `#0D1B2A`**: 텍스트 외곽선/그림자/오버레이 용. 순수 검정(#000)보다 부드럽고 세련된 느낌

### 2.3 사용 규칙

```
텍스트 색상 조합:
  - 메인 텍스트: #FFD600 (노란) + 외곽선 #0D1B2A (네이비)  → 가장 눈에 띔
  - 보조 텍스트: #FFFFFF (흰색) + 외곽선 #0D1B2A
  - 강조 배지:   #2D7FF9 (파랑) 배경 + #FFFFFF 텍스트

배경 오버레이:
  - #0D1B2A, opacity 40-60%  → AI 시연 화면 위에 씌워서 텍스트 가독성 확보
```

## 3. 브랜드 폰트

### 3.1 선정 폰트

| 용도 | 폰트 | 굵기 | 라이선스 |
| --- | --- | --- | --- |
| **메인 텍스트** | Pretendard | ExtraBold (800) | OFL (무료) |
| **보조/영문** | Pretendard | Bold (700) | OFL (무료) |

### 3.2 선정 근거

- **Pretendard**: 한글+영문+숫자 모두 깔끔하게 지원. Apple SF, Inter 스타일의 모던한 산세리프. ExtraBold가 섬네일 텍스트에 적합한 굵기
- 대안: Noto Sans KR Black (Google Fonts), Spoqa Han Sans Neo Bold

### 3.3 텍스트 렌더링 규칙

```
섬네일 텍스트:
  - 최대 5단어 (한글 기준 10자 이내)
  - 폰트 크기: 60-90px (1280x720 기준)
  - 외곽선(stroke): 3-5px, #0D1B2A
  - 그림자: offset (2,2), blur 4px, #000000 50%
  - 모바일 테스트: 360px 폭으로 축소했을 때 여전히 읽히는지 확인
```
