# Step 7: 접근성 & 법적 준수 레이어

> **파이프라인**: 카드뉴스 자동화 도구
> **목적**: 카드뉴스의 접근성 문제와 저작권/라이선스 리스크를 시스템 레벨에서 방지한다.
> **approval_required**: true (법적 리스크가 높은 영역)

---

## 1. 법적 리스크 요약

| 리스크 | 법적 근거 | 처벌 | 시스템 대응 |
| --- | --- | --- | --- |
| **이미지 저작권 침해** | 저작권법 제136조 | 3년 이하 징역 / 3천만원 이하 벌금 | 라이선스 추적 + 안전 소스만 허용 |
| **폰트 라이선스 위반** | 저작권법 / 사용 약관 | 손해배상 청구 | OFL 폰트만 기본 제공 |
| **초상권 침해** | 민법 / 개인정보보호법 | 손해배상 | 인물 사진 경고 시스템 |
| **접근성 미비** | 장애인차별금지법 | 시정 명령 | alt text 자동 생성 |

## 2. 이미지 라이선스 추적 시스템

### 2.1 안전 이미지 소스

| 소스 | 라이선스 | 상업적 사용 | 저작자 표시 |
| --- | --- | --- | --- |
| **Pexels** | Pexels License (CC0 유사) | 가능 | 불필요 (권장) |
| **Unsplash** | Unsplash License | 가능 | 불필요 (권장) |
| **Pixabay** | Pixabay License | 가능 | 불필요 |
| 직접 생성 (AI) | 생성자 소유 | 가능 | 해당 없음 |

### 2.2 라이선스 추적 플로우

```
이미지 사용 시:
  1. LicenseInfo 필수 입력
     {
       type: "cc0" | "free" | "commercial" | "unknown",
       source: "Pexels",
       attribution: "Photo by John",
       url: "https://pexels.com/photo/12345"
     }

  2. 검증 규칙:
     IF type == "unknown":
       경고: "라이선스 미확인 이미지입니다. 사용 전 확인하세요."
       산출물에 [라이선스 검토 필요] 마킹
     IF type == "commercial":
       라이선스 조건 확인 요청

  3. 라이선스 보고서 자동 생성:
     output/license-report.md
     - 사용된 모든 이미지/폰트 목록
     - 각 라이선스 유형, 출처, 저작자
```

### 2.3 금지 행위

```
시스템에서 차단/경고:
  - 출처 불명 이미지 사용 → 경고 + [라이선스 검토 필요]
  - 워터마크가 포함된 이미지 → 감지 시 경고
  - 상업용 폰트 무단 사용 → OFL 아닌 폰트 사용 시 경고
```

## 3. 폰트 라이선스 검증

### 3.1 기본 제공 폰트 (OFL)

| 폰트 | 라이선스 | 파일 |
| --- | --- | --- |
| Pretendard | SIL Open Font License 1.1 | fonts/Pretendard-*.woff2 |
| NanumSquare | SIL Open Font License 1.1 | fonts/NanumSquare-*.woff2 |
| Noto Sans KR | SIL Open Font License 1.1 | fonts/NotoSansKR-*.woff2 |

### 3.2 커스텀 폰트 사용 시

```
사용자가 커스텀 폰트 추가 시:
  1. 라이선스 정보 필수 입력 (FontSpec.license)
  2. 상업적 사용 가능 여부 확인
  3. 라이선스 보고서에 포함
  4. OFL/Apache 아닌 경우 → 경고 메시지
```

## 4. 접근성 (Accessibility)

### 4.1 alt text 자동 생성

```
모든 슬라이드에 alt_text 필수:

생성 규칙:
  cover: "{프로젝트명} 표지. {제목}. {배경 설명}"
  body:  "슬라이드 {N}. {제목}. {본문 요약}. {이미지 설명}"
  closing: "마지막 슬라이드. {CTA 문구}. {브랜드 정보}"

예시:
  "재택근무 생산성 팁 표지. 왜 집에서 더 바쁠까? 노트북과 커피가 있는 책상 이미지."
```

### 4.2 텍스트 전용 버전

```
각 프로젝트에 대해 텍스트 전용 버전 자동 생성:
  output/text-version.md

내용:
  # {프로젝트명}

  ## 슬라이드 1 (표지)
  {제목}
  {부제목}
  [이미지: {alt_text}]

  ## 슬라이드 2
  {제목}
  {본문}
  [이미지: {alt_text}]

  ...
```

### 4.3 색상 대비 검증

```
WCAG 2.1 AA 기준:
  - 일반 텍스트: 대비율 4.5:1 이상
  - 큰 텍스트 (18pt+): 대비율 3:1 이상

검증 방법:
  - 텍스트 색상 vs 배경 색상 대비율 계산
  - 기준 미달 시 경고 + 대안 색상 제안
  - 이미지 위 텍스트 → 오버레이 투명도 조정 제안
```

## 5. 저작권 워터마크 & 출처

```
자동 출처 삽입 옵션:
  - 마지막 장에 이미지 출처 목록 표시 (선택)
  - 브랜드 워터마크 자동 배치 (선택)
  - 라이선스 보고서 별도 생성 (필수)

출처 표기 형식:
  "Images: Pexels (John Doe), Unsplash (Jane Smith)"
```

## 6. 준수 체크리스트

산출물 생성 시 자동 검증:

```
[ ] 모든 이미지에 LicenseInfo 존재
[ ] unknown 라이선스 이미지에 [검토 필요] 마킹
[ ] 모든 폰트에 라이선스 정보 존재
[ ] OFL 아닌 폰트 사용 시 경고
[ ] 모든 슬라이드에 alt_text 존재
[ ] 텍스트 전용 버전 생성
[ ] 텍스트-배경 대비율 3:1 이상
[ ] 라이선스 보고서 생성
```

## 주의사항

- **이미지 저작권은 가장 심각한 법적 리스크** → 시스템에서 반드시 방지
- 나눔스퀘어, 프리텐다드, Noto Sans KR은 OFL 라이선스 (상업적 사용 가능)
- Pexels, Unsplash는 상업적 사용 가능하지만 저작자 표시 권장
- WCAG 2.1 가이드라인 참조 (색상 대비, 텍스트 크기)
- 인물 사진 사용 시 초상권 문제 → 모델 릴리스 확인 안내
