# 스레드 포스팅

---

도메인을 모르면 AI에게 일을 시키기도 어렵다는 걸 깨달았습니다.

"카드뉴스 도구 만들어줘"라고 해도, 판형이 뭔지 CTA가 뭔지 모르면 제대로 된 지시를 내릴 수가 없습니다. 그리고 도메인마다 필요한 개발 단계도 전혀 다릅니다.

그래서 AI가 스스로 도메인을 조사하고, 자기가 따를 매뉴얼을 만들고, 그대로 실행하는 Claude Code 스킬을 만들었습니다.

"카드뉴스 자동화 도구를 만들고 싶어" 한마디에서:
- AI가 웹 검색으로 도메인을 조사하고
- 카드뉴스에 최적화된 8단계 파이프라인을 스스로 설계하고
- 실제 카드뉴스 이미지 7장을 생성했습니다

카드뉴스 도메인 지식은 전혀 필요하지 않았습니다.

표준화된 워크플로우를 AI가 스스로 만들 수 있게 되면, 결과물의 품질도 예측 가능해지고, AI끼리 같은 표준을 공유해서 협업하는 것도 가능해집니다.

자세한 내용은 블로그에 정리했습니다.
