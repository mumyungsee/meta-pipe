# 도메인 테스트

meta-pipe 파이프라인을 특정 도메인으로 테스트하고 결과를 평가합니다.

## 수행 작업

1. `skills/meta-pipe/SKILL.md` 읽기
2. "$ARGUMENTS" 도메인으로 Phase A~C 실행
3. 결과물 품질 자체 평가 (도메인 커버리지, 단계 적절성, 산출물 완성도)
4. 평가 결과를 `test-output/` 에 저장

## 평가 기준

- 도메인 특화도: 생성된 파이프라인이 해당 도메인에 맞는가?
- 단계 완성도: 각 단계의 산출물이 다음 단계 입력으로 충분한가?
- 실용성: 실제 사용 가능한 수준인가?
